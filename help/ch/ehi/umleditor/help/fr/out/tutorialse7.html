3.5 Beziehung

Erstellen Sie eine weitere Klasse Gebaeude.

Erstellen Sie mit Hilfe des Werkzeugs ([8], Kap. 3.4.1) eine Beiehung zwischen den beiden Klassen BoFlaechen und Gebaeude, indem sie auf die Klasse BoFlaechen klicken und ziehen sie den Mauszeiger (linke Maustaste gedrückt halten) auf die Klasse Gebaeude (linke Maustaste loslassen). Sie haben jetzt eine Beziehung zwischen den beiden Klassen erstellt (Abb. 3.8). Geben Sie der Beziehung irgendeinen Namen (in unserem Beispiel GebaeudeFlaeche).


Abbildung 3.8: Eine Beziehung zwischen zwei Klassen.

Im Spezifikationsdialog ([8], Kap. 4.2.11) der beiden Rollen (Abb. 3.9) können Sie ihre Bezeichnungen ändern. Geben Sie für die eine Rolle den Namen Flaeche und für die andere den Namen Gebaeude ein.


Abbildung 3.9: Spezifikationsdialog für eine Rolle.

In den betreffenden Rollen kann man auch die Kardinalität ändern (Abb. 3.10). Stellen Sie für die Rolle Flaeche den Wert 1 und für die Rolle Gebaeude den Wert 0..* ein.


Abbildung 3.10: Kardinalität ändern.

Wenn Sie den Mauszeiger über einer Rolle (Ende der Beziehung positionieren), können Sie mit Hilfe des Kontext-Menus die Beziehung in eine Komposition umwandeln (Abb. 3.11).


Abbildung 3.11: Kontext-Menu einer Rolle.

Um weitere Eigenschaften der Beziehung zu definieren, können Sie die Beziehung markieren und den Spezifikationsdialog öffnen ([8], Kap. 4.2.9).
