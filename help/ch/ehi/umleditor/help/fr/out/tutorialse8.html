3.6 Wertebereich, Einheit

Um einen Wertebereich oder eine Einheit zu definieren, markieren Sie im Navigationsbereich das Paket in dem Sie die Definition anlegen wollen. Mit Hilfe der Funktion Neu/Einheit (im Kontext-Menu) erstellen Sie dann eine entsprechende Definition (Abb. 3.12). Wertebereiche und Einheiten können im UML-Editor nicht im Diagramm dargestellt werden.


Abbildung 3.12: Funktion Neu/Einheit im Navigationsbereich.

Ändern Sie den Namen der Einheit in m. Die exakte Definition der Einheit erfolgt mit Hilfe von INTERLIS. Im Spezifikationsdialog ([8], Kap. 4.2.14) der Einheit (Abb. 3.13) kann man unter Definition einen entsprechenden Text in INTERLIS-Syntax eintragen.
