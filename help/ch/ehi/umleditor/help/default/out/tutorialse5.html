3.3 Löschen

Bitte beachten Sie den Unterschied zwischen Löschen im Modell und Löschen in einem Diagramm: Wenn Sie eine Klasse in einem Diagramm markieren und die Taste Delete drücken, wird die Klasse aus dem Diagramm gelöscht, ist aber im Modell weiterhin enthalten (wie Sie im Navigationsbereich einfach überprüfen können). Um eine Klasse aus dem Modell zu löschen, müssen Sie die rechte Maustaste drücken und die Funktion Löschen (im Modell) verwenden.
